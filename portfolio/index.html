    • Expense Logging : A structured table displays all recorded expenses, making it easy to review spending habits.
    • Delete & Restore Functionality : Users can delete expenses, with the option to restore the most recently deleted entry.

    • The Coin Picker is a simple Python-based tool designed to assist in Housie (Tambola) games. It randomly picks coins (numbers) from a predefined range, ensuring a fair and automated number selection process. Key Features:

    • User-Friendly Interface : Built with Tkinter for easy interaction.
